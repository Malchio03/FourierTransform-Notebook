# Trasformata di Fourier: Elaborazione di Segnali

Questo repository contiene un notebook interattivo (`fourier.ipynb`) sviluppato come materiale per il corso di **Fondamenti dell'Elaborazione di Segnali e Immagini**. Il progetto si concentra sull'applicazione e l'analisi della Trasformata di Fourier Discreta (DFT).

## Panoramica del Progetto
Il notebook esplora i seguenti concetti chiave:
- Creazione e analisi di segnali campionati (funzioni pari e dispari).
- Calcolo della DFT (Discrete Fourier Transform) e visualizzazione dello spettro delle frequenze.
- Spostamento delle semionde (shift) per centrare lo spettro.
- Calcolo della trasformata inversa (IDFT) per ricostruire il segnale originale.

## Requisiti
Il codice è scritto in Python. Per eseguirlo in locale, assicurati di avere installate le seguenti librerie:
- `numpy`
- `matplotlib`
